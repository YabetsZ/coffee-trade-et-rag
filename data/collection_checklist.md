# Coffee trade regulation RAG data collection checklist

Use this checklist to build the Amharic coffee corpus.

## Target documents

- [ ] Coffee trading proclamation
- [ ] Coffee buying and selling rules
- [ ] Coffee grading and quality rules
- [ ] Coffee export or licensing procedure
- [ ] Coffee history and culture
- [ ] Coffee market or value-chain summary

## Where to collect from

- [ ] Ministry of Agriculture Ethiopia
- [ ] Ethiopian Institute of Agricultural Research
- [ ] Ethiopian Coffee and Tea Authority
- [ ] Agricultural Transformation Institute
- [ ] FAO Ethiopia
- [ ] Regional agriculture bureau pages
- [ ] University theses or repository pages
- [ ] Amharic news or agriculture articles

## Suggested workflow

1. Find a PDF, article, or page.
2. Copy only the coffee-related text.
3. Save it as a .txt file in data/corpus/.
4. Give the file a clear name.
5. Keep each file focused on one topic.
6. Add the source URL in a note if needed.

## Good corpus file names

- coffee_trade_law_1.txt
- coffee_trade_law_2.txt
- coffee_trade_law_3.txt
- coffee_trade_law_4.txt
- coffee_trade_summary.txt
- coffee_terms.txt

## Minimum project requirement

A small but acceptable class project can work with 4 clean Amharic coffee regulation PDFs.
